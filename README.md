# Cybersecurity Portfolio (Vite + React + Tailwind)

A single-page portfolio to showcase cybersecurity projects, coursework, and writing. Built with Vite + React, styled with Tailwind, and designed for a dark theme with a mobile-friendly header.
It showcases current projects and papers in a simple layout. 

## Features

Dark UI with centralized theme tokens (THEME)

Sections: Home, About, Experience/Skills, Learning (Courses & Certifications), Portfolio, Writing

Mobile menu with slide-down panel; desktop nav centered

Cards/rows hover with pink-ish tint

Optional snap-to-section scrolling

## Quick Start

Prereqs: Node 18+ and npm

install deps:
npm install

dev server:
npm run dev

production build:
npm run build
